# **Records** 📅
<br/>

## 🚀 Intro

1. 매일 해야할 계획을 저장.
2. 그 계획을 매일 인증
3. 지날 날짜에는 인증을 못하고 당일 00:00 ~ 23:59 에 유효

<br/>
